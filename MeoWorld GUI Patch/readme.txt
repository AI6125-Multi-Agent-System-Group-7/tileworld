Tired of looking at those colour blocks moving around? Lemme introduce you another GUI wrapper: MeoWorld (Tileworld Cat GUI)!

To play with it, just move the assets/ and src/ folder directly to your tileworld project root folder, where the .classpath and .project exists.Note that this will NOT replace or changing any files from the original project, neither touch any existing codes. It does not need any other libraries except reusing the ones already in Tileworld, so make sure you can run the original project before patching.

Then, run the CatGUI with java, you'll get this alternative UI for your project!

Also, it's welcomed to show your creativity by changing it to something like:
- Ninja and scrolls / Developer vs Bugs
- Professor chasing students (idea from Nop)
- etc.
